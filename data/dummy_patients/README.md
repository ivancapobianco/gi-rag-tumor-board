## Dummy Patient Cases

This folder contains synthetic patient cases provided for
demonstration and reproducibility purposes only.

- `example_case_de.txt`  
  Synthetic tumor board case written in German, reflecting
  real-world MDT documentation style.

- `example_case_en_literal.txt`  
  Exact, literal English translation of the German dummy case.
  This file is provided for reviewer readability only.

Important:
The English translation is NOT a rewritten or standardized version.
In the actual study, case rewriting and standardization were
performed automatically using a dedicated prompt-based pipeline.
